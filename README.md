# Countdown

Script count down from starting value

## Formats

You can use several countdown output format:

* One by line
* In one line

## Usage

* `-c` - Count (in seconds)
* `-f` - Format (1 - Every line, 2 - One Line)  

## Examples

One line output with 2 seconds count:

```
./countdown.sh -f 2 -c 2
```

Line by line:

```
./countdown.sh -f 1 -c 2
```
